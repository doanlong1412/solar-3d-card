# 📸 Assets

Đặt ảnh/video demo của bạn vào đây với đúng tên file để README hiển thị đúng:

| File | Dùng ở đâu | Gợi ý |
|---|---|---|
| `preview.gif` | Mục "Demo" trong README | Ghi màn hình ngắn (10–20s) xoay camera quanh villa, đổi giờ ngày/đêm hoặc đổi thời tiết |
| `preview.png` | Mục "Screenshot" trong README | Ảnh chụp toàn cảnh card lúc ban ngày, đẹp và rõ nhất |
| `editor-preview.png` | Mục "Visual Config Editor" trong README | Ảnh chụp trình chỉnh sửa (nhấn ✏️ Edit trên card) |

> File này (`assets/README.md`) chỉ là ghi chú nội bộ — có thể xoá sau khi đã thêm đủ ảnh, không ảnh hưởng đến hoạt động của card.
